# Clochette Android

Base native Android prévue pour donner un vrai corps mobile à Clochette.

La PWA reste le laboratoire de personnalité. Cette partie Android servira à préparer :

- une application Android native ;
- un vrai widget d'écran d'accueil ;
- des notifications locales ;
- une mémoire locale fiable ;
- une future couche Notion optionnelle ;
- une future intégration Gemma locale.

## Principe

Clochette doit fonctionner même sans réseau.

Mémoire locale d'abord :

- phrases adoptées ;
- phrases refusées ;
- carnet ;
- préférences ;
- état du jour.

Notion ensuite, seulement comme mémoire longue volontaire :

- sauvegarde du carnet ;
- base de répliques homologuées ;
- suivi des projets ;
- observations longues.

## Garde-fou

Clochette ne devient pas un tableau de bord.
Elle reste une présence piquante, maternelle, drôle et légèrement envahissante.

Elle formule des hypothèses, pas des verdicts.
